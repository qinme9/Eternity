# Eternity - 永恒模组

为 Minecraft 1.21.1 (NeoForge) 添加一系列"永恒"物品 —— 使用后永不消耗的高品质工具与消耗品。

## 核心特性

所有永恒物品都具有以下属性：
- **堆叠数量为 1**，不可堆叠
- **名称显示为彩虹色渐变**
- **使用后不消耗** —— 食物、药水、投掷物均可无限使用
- 物品栏中显示**附魔光泽**

---

## 物品列表

### 永恒食物（43 种）

覆盖原版几乎所有食物，食用后不消耗。营养值与原版一致。

特殊食物效果标注：

| 物品 | 额外效果 |
|------|---------|
| 永恒腐肉 | 80% 概率获得饥饿 (0:30) |
| 永恒毒马铃薯 | 60% 概率获得中毒 (0:05) |
| 永恒河豚 | 反胃 (0:15) + 饥饿 (0:15) + 中毒 II (1:00) |
| 永恒蜘蛛眼 | 中毒 (0:05) |
| 永恒迷之炖菜 | 每次食用随机获得一种效果 |
| 永恒附魔金苹果 | 生命恢复 II (0:20) + 伤害吸收 IV (2:00) + 抗性提升 (5:00) + 防火 (5:00) |
| 永恒牛奶桶 | 清除所有效果 |
| 永恒紫颂果 | 食用后随机传送 |

### 永恒正面效果食用物品（38 种）

食用后获得 **5 分钟** 持续效果，分为 I 级和 II 级两个版本：

速度、急迫、力量、瞬间治疗、跳跃提升、生命恢复、抗性提升、防火、水下呼吸、隐身、夜视、生命提升、伤害吸收、饱和、发光、幸运、缓降、潮涌能量、海豚的恩惠、村庄英雄

（每个效果提供 I 级和 II 级两个版本）

### 永恒负面效果投掷物品（38 种）

投掷命中后施加 **5 分钟** 负面效果，分为 I 级和 II 级两个版本：

缓慢、挖掘疲劳、瞬间伤害、反胃、失明、饥饿、虚弱、中毒、凋零、飘浮、霉运、黑暗、不祥之兆、袭击之兆、试炼之兆、蓄风、织网、渗浆、虫蚀

### 永恒药水系列（3 种基础类型 × 29 种效果）

| 类型 | 说明 |
|------|------|
| 永恒药水 | 饮用型，可喝下获得对应效果 |
| 永恒喷溅型药水 | 投掷型，撞击时影响范围内生物 |
| 永恒滞留型药水 | 投掷型，撞击时生成效果云持续影响 |

支持所有原版药水效果（夜视、隐身、跳跃提升、抗火、迅捷、缓慢、神龟、水下呼吸、治疗、伤害、中毒、生命恢复、力量、虚弱、缓降、幸运、蓄风、织网、渗浆、虫蚀，含 II 级变体）。

### 永恒不死图腾

- 放在主手或副手，持有者**死亡时自动激活**
- 恢复 1 点生命，清除不良效果
- 施加：生命恢复 II (0:45) + 伤害吸收 II (5:00) + 防火 (5:00)
- 激活后**不消耗**，可反复使用

### 永恒铁砧

- 功能与原版铁砧相同（改名、修理、附魔）
- **永不损坏** —— 不会出现裂纹或毁坏

## 安装要求

- **Minecraft**: 1.21.1
- **模组加载器**: NeoForge
- **Java**: 21+

## 构建方式

```bash
./gradlew build
```

构建产物位于 `build/libs/` 目录。
